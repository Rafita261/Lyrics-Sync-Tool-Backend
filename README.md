# LYRICS SYNC TOOL - BACKEND

C'est le backend de [Lyrics-Sync-Tool](https://github.com/Rafita261/lyrics-sync-tool.git)

## Installation 

- Cloner cette repository

```bash
git clone https://github.com/Rafita261/Lyrics-Sync-Tool-Backend
cd Lyrics-Sync-Tool-Backend
```

- Installer les dépendences

```bash
pip install requirements.txt
```

- Installer **FFmpeg** 

```bash
apt-get update && apt-get install -y ffmpeg
```

## Tester en local

- Exécuter le fichier **app.py** 

```bash
python3 app.py
```

- Tester avec **curl**

```bash
curl -X GET http://127.0.0.1:8080/health
```
